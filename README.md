# Installation

## Quick Start

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/luxaurajs/Luxaura.js)

## Advanced Installation

Clone this repository

````shell
$ cd path/to/repository
$ npm install
$ npm run build
````

If your computer is setup to serve pages, just upload the build (located in the `site` folder) to your public folder and you're done!

**If not:**

````shell
$ npm run serve
````
Your Luxaura.js site is available at http://localhost:5000.
